# Mekanistic Ships

Forge 1.20.1 mod scaffolding targeting Forge 47.4.2.

## Development
- Requires JDK 17 (`JAVA_HOME` set to a Java 17 install).
- Use the Gradle tasks `runClient` and `runServer`.

## Local mod jars
This project references local jars from the PrismLauncher instance provided in `build.gradle`.
